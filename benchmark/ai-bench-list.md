# the top 10 of ai benchmark

- [`Arena`](https://arena.ai/leaderboard/code) - Crowd sourced human evaluation for LLMs
- [`Apex-agents`](https://www.mercor.com/apex/apex-agents-leaderboard/) - Evaluating autonomous agent capabilities
- [`CritPt`](https://artificialanalysis.ai/evaluations/critpt) - Focused on critical thinking and complex reasoning
- [`Humanity's Last Exam`](https://agi.safe.ai/) - A collection of graduate-level expert questions
- [`LiveCodeBench`](https://livecodebench.github.io/leaderboard.html) - Real-time coding benchmark to prevent data contamination
- [`SWE-bench`](https://www.swebench.com/) - Solving real-world GitHub issues
- [`LiveBench`](https://livebench.ai/) - Monthly updated benchmark for reasoning and math
- [`llm-stats`](https://llm-stats.com) - Detailed performance and cost metrics
- [`Onyx`](https://onyx.app/llm-leaderboard) - Multi-modal and agentic workflow evaluation
- [`BenchLM`](https://benchlm.ai/best/overall) - Comprehensive ranking across multiple categories
